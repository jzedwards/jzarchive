Europa Symbol Font

Here at wartwork, we love Game Designers' Workshop (GDW) Europa series. Michael Purcell has created a brilliant symbol font for Europa-style counters. The font is hosted here for use in SVG counters, and as an OTF font file for download and installation. For more info go to the fonts menu link above.

RAF Counters

West End Games peaked with John Butterfield's solitaire RAF in 1986. wartwork have recreated the clasic design using the Inkscape Counter Extension and the source files are available here. New colour counters based on a BoardGameGeek contributor are coming soon, as are reworked cards.

SVG Europa Counters

We have built a spreadsheet which creates sample SVG counters GDW-style based on the Europa font. The spreadsheet has a macro to create files, but if you are concerned about the Excel warning, or want to use the sheet in Open Office or Google Docs you can save the counter files one by one. Good luck with that...

The SPI Archive simpubs

Our sister site is simpubs.org which is an archive of classic 1970s and 80s SPI game scans.
