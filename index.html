Welcome to jzedwards github website.

This site is a github web page of content by jzedward. The main content is hosted font files for NATO symbols, to enable shared SVG files to make use of custom fonts.

There is also a symbol font for GDW Europa by Michael Purcell available here.

This page uses (free) Tex Gyre Heros font to mimic Helvetica from CDNfonts. This font also features a condensed version.
